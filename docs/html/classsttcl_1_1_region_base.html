template<class StateMachineImpl, class StateInterface, class EventArgs = void>
typedef void(StateInterface::* sttcl::RegionBase< StateMachineImpl, StateInterface, EventArgs >::EventHandler)(StateMachineImpl *, const EventArgs *)

The inner states event handler signature.

All methods of the IInnerState interface must have this signature. The first parameter is a pointer to the containing state machine. The second parameter is a pointer to additional event arguments as specified with the EventArgs template parameter.
